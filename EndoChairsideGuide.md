# Endodontic Chairside Guide

## Before Starting
- Pre-op PA/BW reviewed
- CBCT reviewed if available
- Estimated chamber depth: ___ mm
- Estimated WL: ___ mm
- Rubber dam/isolation confirmed
- Irrigants ready
- EAL ready
- 10C files ready
- Orifice opener / guide path / .04 files ready

---

## Phase 1 — Access
Goal: Safely reach chamber and identify canals.

Checklist:
[ ] Mark 557 bur to target depth
[ ] Access chamber
[ ] Confirm chamber with DG16/17
[ ] Refine with Endo-Z
[ ] Identify canals

Stop rule:
If marked depth reached but chamber not found:
→ Stop, radiograph, reassess angulation.

---

## Phase 2 — Initial Scouting
Goal: Determine available treatment space.

[ ] Lubricant or aqueous irrigation
[ ] 10C file to passive resistance
[ ] Do not exceed estimated WL
[ ] Measure available treatment space

Decision:
- Reaches estimated WL → proceed to orifice opening / EAL
- Stops short but >16 mm → proceed with caution
- ≤16 mm → high difficulty; consider referral

Record:
Available treatment space: ___ mm

---

## Phase 3 — Working Length
Goal: Establish EAL 0, patency length, shaping length.

[ ] Dry canal
[ ] Attach EAL to 10C
[ ] Advance until EAL 0
[ ] Confirm with WL radiograph
[ ] Record reference point

Record:
EAL 0 / Guide path length: ___ mm  
Patency length: ___ mm  
Shaping length: ___ mm  
Reference point: ___

---

## Phase 4 — Glide Path
Goal: 10C super loose at patency length, guide path file to EAL 0.

[ ] 10C to patency length until super loose
[ ] Irrigate
[ ] Guide path file to EAL 0
[ ] Irrigate

If guide path file does not reach length:
→ Return to 10C patency work.

---

## Phase 5 — Shaping
Goal: Create final .04 taper shape.

[ ] Gauge with 25 NiTi hand file at shaping length
[ ] Select matching .04 engine-driven file
[ ] Shape to shaping length
[ ] Irrigate and recapitulate
[ ] Aim for at least size 30/.04 when possible

Record:
Final rotary file: ___ /.04  
Shaping length: ___ mm

---

## Phase 6 — Smear Layer and Final Disinfection
[ ] 17% EDTA 90–120 sec
[ ] Agitate with 30/.04 GP cone, 2–3 mm motion
[ ] Concentrated NaOCl final disinfection

---

## Phase 7 — Cone Fit
[ ] Gauge obturation with NiTi hand file
[ ] Select matching .04 GP cone
[ ] Fit to shaping length
[ ] Confirm radiographically

Record:
Master cone: ___ /.04  
Cone fit radiograph: acceptable / adjust

---

## Phase 8 — Dry, Seal, Seat
[ ] Dry with 25/.04 paper points
[ ] Confirm dry/slightly damp
[ ] Apply bioceramic sealer with White NaviTip
[ ] Paper point through sealer to shaping length
[ ] Re-apply sealer
[ ] Seat pre-fit GP cone

---

## Phase 9 — Downpack / Backfill / Closure
[ ] Sear or downpack as indicated
[ ] Backfill remaining canal space
[ ] Compact lightly
[ ] Clean chamber
[ ] Rinse chamber
[ ] Close access

Closure:
[ ] Sponge + temp
[ ] Orifice barrier + temp
[ ] Final restoration
