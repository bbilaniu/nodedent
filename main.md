# Basic Endodontic technique treatment algorithm {#basic-endodontic-technique-treatment-algorithm}

## ESTIMATE TARGET VERTICAL DEPTH {#estimate-target-vertical-depth}

\==================================================

### STEP 1 {#step-1}

\==================================================  
ESTIMATE TARGET VERTICAL DEPTH to the pulp chamber from the preoperative bitewing radiograph.  
Use CBCT if available.

\==================================================

### STEP 2 {#step-2}

\==================================================  
Measure and mark the target vertical access depth on the **557 bur**.

## ACCESS PULP CHAMBER {#access-pulp-chamber}

\==================================================

### STEP 3 {#step-3}

\==================================================  
ACCESS PULP CHAMBER using a **557 bur**.  
Rinse access with water as needed.  
A. **557 bur** reaches the pulp chamber, [GOTO STEP 4](#step-4).  
B. **557 bur** reaches the marked target vertical depth but not the pulp chamber, **STOP**  
1\. Capture bitewing radiograph (posterior teeth) and/or periapical radiograph (anterior teeth).  
2\. Evaluate radiographs and clinically assess orientation relative to the long axis of the tooth.  
3\. Redirect access direction accordingly.

\==================================================

### STEP 4  {#step-4}

\==================================================  
Confirm access to the pulp chamber using a **DG16/17 Endo Explorer**.

\==================================================

### STEP 5 {#step-5}

\==================================================  
Cut and refine access outline form with **Endo-Z bur**.  
Rinse access with water as needed.

\==================================================

### STEP 6 {#step-6}

\==================================================  
Identify canal(s) using **DG16/17 Endo Explorer**.

## ESTIMATE WORKING LENGTH  {#estimate-working-length}

\==================================================

### STEP 7 {#step-7}

\==================================================  
ESTIMATE WORKING LENGTH from the preoperative periapical radiograph.  
Use CBCT if available.

\==================================================

### STEP 8  {#step-8}

\==================================================  
Set file stopper to the estimated working length on a size **10C hand file**.

\==================================================

### STEP 9 {#step-9}

\==================================================  
Place **file lubricant** or **aqueous irrigation** into the canal.

## ASSESS AVAILABLE TREATMENT SPACE. {#assess-available-treatment-space}

\==================================================

### STEP 10 {#step-10}

\==================================================  
ASSESS AVAILABLE TREATMENT SPACE. Advance the **10C hand file** to passive resistance.  
Do not exceed estimated working length.  
A. If the file reaches the estimated working length → [GOTO STEP 13](#step-13).  
B. If the file stops short of the estimated working length → [GOTO STEP 11](#step-11).

\==================================================

### STEP 11 {#step-11}

\==================================================  
With the **10C hand file** in the canal at its terminal position, adjust the file stopper to a reproducible reference point, remove the file from the canal.

\==================================================

### STEP 12  {#step-12}

\==================================================  
Measure the available treatment space.  
A. File advances \>16 mm → [GOTO STEP 13](#step-13).  
B. File advances ≤16 mm. CASE DIFFICULTY INCREASES EXPONENTIALLY. REASSESS ABILITY TO PROCEED.  
1\. If comfortable → [GOTO STEP 13](#step-13) WITH EXTREME CAUTION.  
2\. If case difficulty exceeds skill limits → [GOTO STEP 80](#step-80).

## OPEN THE CANAL ORIFICE {#open-the-canal-orifice}

\==================================================

### STEP 13 {#step-13}

\==================================================  
OPEN THE CANAL ORIFICE with a dedicated engine-driven **orifice opener file**.  
Advance the orifice opener into the canal. Do not exceed 15 mm.

\==================================================

### STEP 14 {#step-14}

\==================================================  
Irrigate with diluted sodium hypochlorite (NaOCl).

\==================================================

### STEP 15 {#step-15}

\==================================================  
Set the file stopper to the estimated working length on a **10C hand file**.

\==================================================

### STEP 16 {#step-16}

\==================================================  
Place a **file lubricant** or **aqueous irrigation in the canal.** 

### STEP 17 {#step-17}

\==================================================  
Advance the **10C hand file toward estimated working length.**  
Use watch-winding and balanced-force motions as needed.  
A. If the file reaches estimated working length → [GOTO STEP 20](#step-20).  
B. If the file does not reach estimated working length → [GOTO STEP 18](#step-18).

\==================================================

### STEP 18 {#step-18}

\==================================================  
With the **10C hand file** in the canal at its terminal position, adjust the file stopper to a reproducible reference point and remove the file from the canal.

## REASSESS AVAILABLE TREATMENT SPACE {#reassess-available-treatment-space}

\==================================================

### STEP 19 {#step-19}

\==================================================  
REASSESS AVAILABLE TREATMENT SPACE. Measure the 10C hand file. This measurement becomes the new available treatment space.  
A. File advances to within 0–3 mm of the estimated WL, [GOTO STEP 20](#step-20).  
B. File advances \> 3 mm from the estimated working length. CASE DIFFICULTY INCREASES. REASSESS ABILITY TO PROCEED.  
1\. If comfortable [GOTO STEP 20](#step-20) WITH CAUTION.  
2\. If case difficulty exceeds skill limits, [GOTO STEP 80](#step-80).

\==================================================

### STEP 20  {#step-20}

\==================================================  
Irrigate with diluted sodium hypochlorite (NaOCl)

\==================================================

### STEP 21  {#step-21}

\==================================================  
Dry canal with **size 25/.04 paper points**.

\==================================================

### STEP 22 {#step-22}

\==================================================  
Attach an **Electronic Apex Locator (EAL)** to the 10C hand file and attempt to establish electronic working length.  
A. **10C hand file** advances and the **EAL** signals patency (audible and/or visual), [GOTO STEP 31](#step-31).  
B. **10C hand file** is advanced to its terminal length and the **EAL** signals “short”, [GOTO STEP 23](#step-23).

\==================================================

### STEP 23 {#step-23}

\==================================================  
Determine **FILE RESISTANCE** \-vs- **FILE STOP**.  
A. File resistance, [GOTO STEP 26](#step-26).  
B. File stop, [GOTO STEP 24](#step-24).

\==================================================

### STEP 24  {#step-24}

\==================================================  
Place a **file lubricant** or **aqueous irrigation in the canal.**

\==================================================

### STEP 25 {#step-25}

\==================================================  
Use file bending technique with **10C hand file**.  
A. File advances, [GOTO STEP 21](#step-21).  
B. File does not advance. CASE DIFFICULTY INCREASES SIGNIFICANTLY. REASSESS ABILITY TO PROCEED.  
1\. If comfortable, [GOTO STEP 24](#step-24).  
2\. If case difficulty exceeds skill limits, [GOTO STEP 80](#step-80).

\==================================================

### STEP 26 {#step-26}

\==================================================  
Place a **file lubricant** or **aqueous irrigation** in the canal.

\==================================================

### STEP 27 {#step-27}

\==================================================  
Open the middle third of the canal with a dedicated engine driven **Guide Path File**.  
Advance the **Guide Path File** to resistance. Do not exceed the available treatment space length from STEP 19\.

\==================================================

### STEP 28 {#step-28}

\==================================================  
Irrigate with diluted NaOCl.

\==================================================

### STEP 29 {#step-29}

\==================================================  
Dry canal with **size 25/.04 paper points**.

\==================================================

### STEP 30 {#step-30}

\==================================================  
[GOTO STEP 22](#step-22).

## ESTABLISHING WORKING LENGTHS {#establishing-working-lengths}

\==================================================

### STEP 31 {#step-31}

\==================================================  
ESTABLISHING WORKING LENGTHS. Adjust the **10C hand file** in the canal until the **EAL** reads “0” (this is EAL 0).

\==================================================

### STEP 32 {#step-32}

\==================================================  
Adjust the **10C hand file** stopper to a reproducible reference point.

\==================================================

### STEP 33 {#step-33}

\==================================================  
Capture a working length radiograph and visually confirm length.

\==================================================

### STEP 34 {#step-34}

\==================================================  
Remove and measure the **10C hand file**. This measurement becomes the EAL 0 measurement.

\==================================================

### STEP 35 {#step-35}

\==================================================  
Record the following information:

***EAL 0 \= Guide Path Length***  
***EAL 0 \+ 1mm \= EAL \+1 \= Patency Length***  
***EAL 0 \- 1mm \= EAL \-1 \= Patency Length***  
Canal location  
Reproducible reference point used.

\==================================================

### STEP 36 {#step-36}

\==================================================  
Place a **file lubricant** or **aqueous irrigation** in the canal.

\==================================================

### STEP 37 {#step-37}

\==================================================  
Measure a **10C hand file** to ***Patency Length*** and advance the file to the predetermined reference point.  
Work the **10C hand file** at ***Patency Length*** in a filing and reciprocating motion until the **10C hand file** becomes SUPER LOOSE.  
A. 10C hand file achieves ***Patency Length***, [GOTO STEP 38](#step-38).  
B. 10C hand file stops short of ***Patency Length***, [GOTO STEP 24](#step-24).

\==================================================

### STEP 38 {#step-38}

\==================================================  
Irrigate with diluted NaOCl.

## CREATE A GUIDE PATH {#create-a-guide-path}

\==================================================

### STEP 39 {#step-39}

\==================================================  
CREATE A GUIDE PATH. Measure a dedicated engine driven **Guide Path File** to ***Guide Path Length (EAL 0\)*** and advance.  
A. **Guide Path File** advances to ***Guide Path Length***, [GOTO STEP 40](#step-40).  
B. **Guide Path File** does not advance to ***Guide Path Length***, [GOTO STEP 36](#step-36).

\==================================================

### STEP 40 {#step-40}

\==================================================  
Irrigate with diluted NaOCl.

## GAUGE FOR FINAL SHAPE {#gauge-for-final-shape}

\==================================================

### STEP 41 {#step-41}

\==================================================  
GAUGE FOR FINAL SHAPE. Set the stopper to **Shaping Length (EAL \-1)** on a **25 NiTi hand file**. Advance the file to ***Shaping Length***.  
A. File advances to ***Shaping Length*** with no resistance, [GOTO STEP 42](#step-42).  
B. File advances to within 0-2mm of ***Shaping Length*** with resistance, [GOTO STEP 43](#step-43).  
C. File advances \> 2mm short of ***Shaping Length***, [GOTO STEP 37](#step-37).

\==================================================

### STEP 42 {#step-42}

\==================================================  
Set the stopper to ***Shaping Length*** on the **next successively larger NiTi hand file**. Advance the file to ***Shaping Length***.  
A. File advances to ***Shaping Length*** with no resistance, [GOTO STEP 42](#step-42).  
B. File advances to within 0-2mm of ***Shaping Length*** with resistance, [GOTO STEP 43](#step-43).

\==================================================

### STEP 43 {#step-43}

\==================================================  
Irrigate with diluted NaOCl.

\==================================================

### STEP 44 {#step-44}

\==================================================  
Place a **file lubricant** or **aqueous irrigation** in the canal.

## CREATE THE FINAL SHAPE {#create-the-final-shape}

\==================================================

### STEP 45 {#step-45}

\==================================================  
CREATE THE FINAL SHAPE. Choose a **.04 tapered engine driven file** matching the same tip size as the **NiTi hand file** that met resistance in STEP 41 and/or STEP 42\. Set the stopper on the **.04 tapered engine driven file** to ***Shaping Length***. Advance the .**04 tapered engine driven file** in the canal.  
A. **.04 tapered engine driven file** advances to ***Shaping Length***. [GOTO STEP 50](#step-50).  
B. **.04 tapered engine driven file** does not advance to ***Shaping Length***. [GOTO STEP 46](#step-46).

\==================================================

### STEP 46 {#step-46}

\==================================================  
Irrigate with diluted NaOCl.

\==================================================

### STEP 47 {#step-47}

\==================================================  
Measure **10C hand file** to ***Patency Length***and advance the file to the predetermined reference point. Work the **10C hand file** at ***Patency Length*** in a filing and reciprocating motion until the **10C hand file** becomes **SUPER LOOSE**.  
A. Size **10C hand file** achieves ***Patency Length***, [GOTO STEP 48](#step-48).  
B. Size **10C hand file** does not achieve ***Patency Length***, [GOTO STEP 24](#step-24).

\==================================================

### STEP 48 {#step-48}

\==================================================  
Irrigate with diluted NaOCl.

\==================================================

### STEP 49 {#step-49}

\==================================================  
Set stopper on the n**ext successively smaller .04 tapered engine driven file** to ***Shaping Length***.  
A. .**04 tapered engine driven file** advances to ***Shaping Length***, [GOTO STEP 50](#step-50).  
B. .**04 tapered engine driven file** does not advance to ***Shaping Length***, [GOTO STEP 46](#step-46).

\==================================================

### STEP 50 {#step-50}

\==================================================  
Evaluate the size of the **.04 tapered engine driven file** that achieved ***Shaping Length***.  
A. **.04 tapered engine driven file** is ≥ to a size 30, [GOTO STEP 55](#step-55).  
B. **.04 tapered engine driven file** is \< than a size 30, [GOTO STEP 51](#step-51).

\==================================================

### STEP 51 {#step-51}

\==================================================  
Irrigate with diluted NaOCl.

\==================================================

### STEP 52 {#step-52}

\==================================================  
Measure the **10C hand file** to ***Patency Length*** and advance the file to the predetermined reference point. Work the **10C hand file** at ***Patency Length*** in a filing and reciprocating motion until the **10C hand file** becomes SUPER LOOSE.

\==================================================

### STEP 53 {#step-53}

\==================================================  
Irrigate with diluted NaOCl.

\==================================================

### STEP 54 {#step-54}

\==================================================  
Increase the size of the final shape preparation to a **size 30/.04 tapered engine driven** **instrument**. Beginning with the size of the **.04 tapered engine driven file** that achieved ***Shaping Length***, set the stopper on the **next successively larger .04 tapered engine driven file** to ***Shaping Length*** and advance the file to ***Shaping Length***.  
A. **04 tapered engine driven file** advances to ***Shaping Length*** [GOTO STEP 50](#step-50).  
B. .**04 tapered engine driven file** does not advance to ***Shaping Length*** [GOTO STEP 41](#step-41).

\==================================================

### STEP 55 {#step-55}

\==================================================  
Irrigate with diluted NaOCl.

\==================================================

### STEP 56 {#step-56}

\==================================================  
Measure the **10C file** to ***Patency Length*** and advance the file to the predetermined reference point. Work the **10C file** at ***Patency Length*** in a filing and reciprocating motion until the **10C file** becomes SUPER LOOSE.

## REMOVE SMEAR LAYER. {#remove-smear-layer}

\==================================================

### STEP 57 {#step-57}

\==================================================  
REMOVE SMEAR LAYER. Irrigate with 17% EDTA. Ideal canal exposure to 17% EDTA \= 90-120 seconds.

\* ***Remove Calcium Hydroxide*** \- prompted from STEP 82\.

\==================================================

### STEP 58 {#step-58}

\==================================================  
Agitate the 17% EDTA in the canal. Measure a **size 30/.04 tapered gutta-percha cone** to ***Shaping Length***. Place **size 30/.04 tapered gutta-percha cone** in the canal and pump the **size 30/.04 tapered gutta-percha cone** up and down within a 2-3mm envelope of motion. DO NOT EXCEED SHAPING LENGTH. GUTTA-PERCHA CONE MAY NOT REACH SHAPING LENGTH.

## FINAL DISINFECTION. {#final-disinfection}

\==================================================

### STEP 59 {#step-59}

\==================================================  
FINAL DISINFECTION. Irrigate with concentrated NaOCl and leave it in the canal.

## GAUGE FOR OBTURATION {#gauge-for-obturation}

\==================================================

### STEP 60 {#step-60}

\==================================================  
GAUGE FOR OBTURATION. Set the file stopper to the ***Shaping Length*** on a **size 30 NiTi hand file**. Advance the **size 30 NiTi hand file** to ***Shaping Length*** and apply firm pressure.  
A. **Size 30 NiTi hand file** achieves ***Shaping Length*** and does not advance under firm apical pressure. [GOTO STEP 64](#step-64).  
B. **Size 30 NiTi hand file** achieves ***Shaping Length*** and advances beyond Shaping Length under firm apical pressure. [GOTO STEP 63](#step-63).  
C. **Size 30 NiTi hand file** does not achieve **Shaping Length**, [GOTO STEP 61](#step-61).

\==================================================

### STEP 61 {#step-61}

\==================================================  
Set the file stopper to ***Shaping Length*** on a **size 25 NiTi hand file**. Advance the **size 25 NiTi hand file** to ***Shaping Length*** and apply form apical pressure.  
A. **Size 25 NiTi hand file** achieves ***Shaping Length*** and does not advance under pressure. [GOTO STEP 64](#step-64).  
B. Size 25 NiTi hand file achieves ***Shaping Length*** and advances beyond Shaping Length under pressure. [GOTO STEP 63](#step-63).  
C. Size 25 NiTi hand file does not achieve ***Shaping Length***, [GOTO STEP 62](#step-62).

\==================================================

### STEP 62 {#step-62}

\==================================================  
Measure **10C hand file** to ***Patency Length*** and advance the file to the predetermined reference point. Work the **10C hand file** at ***Patency Length*** in a filing and reciprocating motion until the 10C hand file becomes SUPER LOOSE.  
A. Size **10C hand file** achieves ***Patency Length***, [GOTO STEP 39](#step-39).  
B. Size **10C hand file** does not achieve ***Patency Length***, [GOTO STEP 25](#step-25).

\==================================================

### STEP 63 {#step-63}

\==================================================  
Set the stopper on the **next successively larger NiTi hand file** to ***Shaping Length***. Advance the NiTi file to ***Shaping Length*** and apply firm apical pressure.  
A. **NiTi File** achieves ***Shaping Length*** and does not advance under firm apical pressure. [GOTO STEP 64](#step-64).  
B. **NiTi File** achieves ***Shaping Length*** and advances beyond ***Shaping Length*** under firm apical pressure. [GOTO STEP 63](#step-63).

\==================================================

### STEP 64 {#step-64}

\==================================================  
Record the size of the **NiTi hand file** that achieved Shaping Length and did not advance under firm apical pressure.

\==================================================

### STEP 65 {#step-65}

\==================================================  
Irrigate with concentrated NaOCl.

## GUTTA-PERCHA CONE FITTING. {#gutta-percha-cone-fitting}

\==================================================

### STEP 66 {#step-66}

\==================================================  
GUTTA-PERCHA CONE FITTING. Choose and measure a **.04 tapered gutta-percha cone** of the same tip size as recorded in STEP 64 to ***Shaping Length***. Place the **.04 tapered gutta-percha cone** in the canal to ***Shaping Length***.  
A. **.04 tapered gutta-percha cone** advances to ***Shaping Length*** [GOTO STEP 73](#step-73).  
B. **.04 tapered gutta-percha cone** does not advance to ***Shaping Length*** [GOTO STEP 67](#step-67).  
C. **.04 tapered gutta-percha cone** advances beyond ***Shaping Length*** [GOTO STEP 68](#step-68).

\==================================================

### STEP 67 {#step-67}

\==================================================  
Choose and measure a **.04 tapered gutta-percha cone** of *one size smaller* than that used in STEP 66\.  
A. **.04 tapered gutta-percha cone** advances to ***Shaping Length*** [GOTO STEP 73](#step-73).  
B. **.04 tapered gutta-percha cone** does not advance to ***Shaping Length*** [GOTO STEP 45](#step-45).  
C. **.04 tapered gutta-percha cone** advances beyond ***Shaping Length*** [GOTO STEP 68](#step-68).

\==================================================

### STEP 68 {#step-68}

\==================================================  
With the .**04 tapered gutta-percha cone** at its terminal apical position, grasp the **.04 tapered gutta-percha cone** with a l**ocking cotton forceps** and crimp the **.04 tapered gutta-percha cone** at the level of the recorded reference point.

\==================================================

### STEP 69 {#step-69}

\==================================================  
While engaged in the **locking cotton forceps**, remove the **.04 tapered gutta-percha cone** from the canal and measure the length.

\==================================================

### STEP 70 {#step-70}

\==================================================  
Subtract the ***Shaping Length*** from **engaged .04 tapered gutta-percha cone** length.

\==================================================

### STEP 71 {#step-71}

\==================================================  
Using a **scissors**, trim the excess length from the **engaged .04 tapered gutta-percha cone**. (The difference between the length of the **engaged .04 tapered gutta-percha cone** and the ***Shaping Length***.)

\==================================================

### STEP 72 {#step-72}

\==================================================  
Placed the **Trimmed, engaged .04 tapered gutta-percha cone** back in the canal to ***Shaping Length***.  
	A. **Trimmed, engaged .04 tapered gutta-percha cone** advances to ***Shaping Length*** [GOTO STEP 73](#step-73).  
	B. **Trimmed, engaged .04 tapered gutta-percha cone** does not advance to ***Shaping Length*** [GOTO STEP 67](#step-67).  
	C. **Trimmed, engaged .04 tapered gutta-percha cone** advances beyond ***Shaping Length***, [GOTO STEP 68](#step-68).

\==================================================

### STEP 73 {#step-73}

\==================================================  
Confirm gutta-percha cone fit with a periapical radiograph.

\==================================================

### STEP 74 {#step-74}

\==================================================  
Remove the **.04 tapered gutta-percha point** from the canal and place it aside for later use. (Be sure to record the proper canal location in multi-rooted teeth).

\==================================================

### STEP 75 {#step-75}

\===================================  
Irrigate with concentrated NaOCl.

\==================================================

### STEP 76 {#step-76}

\==================================================  
Agitate concentrated NaOCl. Measure a **size 30/.04 tapered gutta-percha cone** to ***Shaping Length.*** Place the **size 30/.04 tapered gutta-percha cone** in the canal and pump the **size 30/.04 tapered gutta-percha cone** up and down within a 2-3mm envelope of motion. DO NOT EXCEED *SHAPING LENGTH*.

\==================================================

### STEP 77 {#step-77}

\==================================================  
Irrigate with concentrated NaOCl.

\==================================================

### STEP 78 {#step-78}

\==================================================  
Remove excess NaOCl from the canal by applying back-pressure on an empty irrigation syringe.

## DRY THE CANAL.  {#dry-the-canal}

\==================================================

### STEP 79 {#step-79}

\==================================================  
DRY THE CANAL. Measure **size** **25/.04 tapered paper points** to the ***Shaping Length***. Advance the size 25/.04 tapered paper points to ***Shaping Length***. Remove the paper point and inspect for moisture. (NOTE, larger paper points can be used for larger canals initially, but canal drying should be honed with **size 25/.04 tapered paper points**).  
	A. Paper point is dry to slightly damp upon removal at ***Shaping Length***, [GOTO STEP 83](#step-83).  
	B. Paper point is wet upon removal at Shaping Length, [GOTO STEP 79](#step-79).  
	C. Paper point remains wet following several attempts to dry, [GOTO STEP 81](#step-81).

\==================================================

### STEP 80 {#step-80}

\==================================================  
Dry the canal with **size 25/.04 taper paper points**.

\==================================================

### STEP 81 {#step-81}

\==================================================  
Place Calcium Hydroxide with a **White NaviTip** into the canal.

\==================================================

### STEP 82 {#step-82}

\==================================================  
CLOSE THE ACCESS with sponge and temporary restorative material.  
	A. If this pathway was reached via STEP 79 begin with [STEP 57](#step-57) at next visit.  
B. If this pathway was reached via any other STEP above, consider reevaluation of case difficulty. Refer if uncomfortable continuing treatment.

\==================================================

### STEP 83 {#step-83}

\==================================================  
Measure the **10C hand file** to Patency Length and advance the file to the predetermined reference point. Work the **10C hand file** at Patency Length in a filing and reciprocating motion until the **10C hand file** becomes SUPER LOOSE.

## APPLY SEALER INTO THE CANAL.  {#apply-sealer-into-the-canal}

\==================================================

### STEP 84 {#step-84}

\==================================================  
APPLY SEALER INTO THE CANAL. Place a **White NaviTip** on the **injectable bioceramic sealer** tube. Measure the **White NaviTip** to 15mm. Place the White NaviTip in the canal to 15mm passively. Inject the **bioceramic sealer** while slowly withdrawing the **White NaviTip** in a coronal direction completely filling the canal space.   
DO NOT BIND THE **White NaviTip** IN THE CANAL.  
For canals with ***Shaping Lengths*** \< 18mm, DO NOT ADVANCE the **White NaviTip** INTO THE LAST 3MM OF THE PREPARED CANAL, as determined by the ***Shaping Length***.

\==================================================

### STEP 85 {#step-85}

\==================================================  
Measure a **size** **25/.04 tapered paper point** to the ***Shaping Length***. Advance the **size 25/.04 tapered paper point** through the applied sealer to ***Shaping Length***.

## RE-APPLY SEALER INTO THE CANAL.  {#re-apply-sealer-into-the-canal}

\==================================================

### STEP 86 {#step-86}

\==================================================  
RE-APPLY SEALER INTO THE CANAL. Place a **White NaviTip** on the **injectable bioceramic sealer**. Measure the **White NaviTip** to 15mm. Place the **White NaviTip** in the canal to 15mm passively. Inject the **bioceramic sealer** while slowing moving the White NaviTip in a coronal direction completely filling the canal space.  
DO NOT BIND **White NaviTip** IN THE CANAL.  
For canals with ***Shaping Lengths*** \< 18mm, DO NOT ADVANCE the **White NaviTip** INTO THE LAST 3MM OF THE PREPARED CANAL, as determined by the ***Shaping Length***.

## GUTTA-PERCHA CONE SEATING.  {#gutta-percha-cone-seating}

\==================================================

### STEP 87 {#step-87}

\==================================================  
GUTTA-PERCHA CONE SEATING. Advance pre-fit **.04 tapered gutta-percha point** from STEP 7 — ESTIMATE WORKING LENGTH4 into the canal and advance it to ***Shaping Length***.  
	A. **.04 tapered gutta-percha point** seats at ***Shaping Length***. [GOTO STEP 88](#step-88).  
	B. **.04 tapered gutta-percha point** does not seat to ***Shaping Length***. [GOTO STEP 89](#step-89).  
	C. **.04 tapered gutta-percha point** advances beyond ***Shaping Length***, [GOTO STEP 60](#step-60).

\==================================================

### STEP 88 {#step-88}

\==================================================  
Measure the **NiTi hand file** recorded in STEP 64 to ***Shaping Length***. Advance the **NiTi hand file** to ***Shaping Length***. Work the **NiTi hand file** at Shaping Length in a reciprocating motion until the **NiTi hand file** becomes SUPER LOOSE. [GOTO STEP 86](#step-86).

\==================================================

### STEP 89 {#step-89}

\==================================================  
Evaluate orifice gap space.  
	A. No gap space present, [GOTO STEP 92](#step-92).  
	B. Gap space present in "round" orifice. [GOTO STEP 90](#step-90).  
	C. Gap space present in "ovoid" orifice. [GOTO STEP 91](#step-91).

## MODIFIED GUTTA-PERCHA DOWN-PACK.  {#modified-gutta-percha-down-pack}

\==================================================

### STEP 90 {#step-90}

\==================================================  
MODIFIED GUTTA-PERCHA DOWN-PACK. Using a dedicated **electric heat source**, measure a **heat plugger** to 15mm. Activate the heat source and while activated, advance the **heat plugger** to the predetermined reference point. While at the 15mm level, deactivate the heat source and wait 5 seconds. After 5 seconds, reactivate heat source and remove the heat plugger from the canal.  
	A. Coronal section of gutta-percha is removed with the **heat plugger** and the remaining gutta-percha stays in place, [GOTO STEP 94](#step-94).  
	B. Little to no gutta-percha is removed from the canal with the **heat plugger**. Increase the size of the **heat plugger**, [GOTO STEP 90](#step-90).  
	C. The gutta-percha cone moves in a coronal direction and/or is removed altogether when the **heat plugger** is withdrawn. Attempt to reseat the gutta-percha cone. If the gutta-percha cone can not be reseated accurately, [GOTO STEP 66](#step-66).  
\* Consider a radiograph to verify proper placement.

\==================================================

### STEP 91 {#step-91}

\==================================================  
Place additional **size 25/.04 tapered gutta-percha points** in the visible gap space(s). Use as many as needed, until there is no visible gap space, [GOTO STEP 92](#step-92).

## SEAR GUTTA-PERCHA CONES. {#sear-gutta-percha-cones}

\==================================================

### STEP 92 {#step-92}

\==================================================  
SEAR GUTTA-PERCHA CONES. Using a dedicated **electric heat source** and **heat plugger**, activate the **heat source**. While activated, use the tip of the **heat plugger** to heat sear the gutta-percha cone at and above the level of the canal orifice.

\==================================================

### STEP 93 {#step-93}

\==================================================  
Using an **endodontic plugger**, apply light vertical compaction force on the remaining gutta-percha in the canal.  
	A. Gutta-percha maintains its current position in the canal (no movement), [GOTO STEP 97](#step-97).  
	B. Gutta-percha moves in the apical direction, [GOTO STEP 94](#step-94).

\==================================================

### STEP 94 {#step-94}

\==================================================  
Re-apply sealer into the canal. Place a **White NaviTip** on the **injectable bioceramic sealer** and inject a small amount on top of the visible gutta-percha.

## BACKFILL THE REMAINDER OF THE CANAL SPACE {#backfill-the-remainder-of-the-canal-space}

\==================================================

### STEP 95 {#step-95}

\==================================================  
BACKFILL THE REMAINDER OF THE CANAL SPACE. An injectable thermoplastic gutta-percha delivery system is turned on and allowed to reach full temperature.

Once at full temperature, the tip of the delivery system is applied through the sealer until it is seated on the remaining gutta-percha in the canal. Keep the tip of the thermoplastic gutta-percha delivery system in contact with the remaining gutta-percha for 5 seconds. Inject the thermoplastic gutta-percha in the canal and allow it to passively withdraw from the canal.

\==================================================

### STEP 96 {#step-96}

\==================================================  
Using an endodontic plugger, apply light vertical compaction force on the injected gutta-percha in the canal.  
	A. Gutta-percha remains exactly at the canal orifice, [GOTO STEP 97](#step-97).  
B. Gutta-percha moves in the apical direction, [GOTO STEP 94](#step-94).  
C. Excess gutta-percha extends into the pulp chamber, [GOTO STEP 97](#step-97).  
D. Plugger "pokes holes" in the injected gutta-percha, increase the size of the endodontic plugger and [GOTO STEP 95](#step-95) if needed.

\==================================================

### STEP 97 {#step-97}

\==================================================  
Remove residual gutta-percha from the pulp chamber with a small spoon excavator and dedicated electric heat source as needed.

\==================================================

### STEP 98 {#step-98}

\==================================================  
Rinse the pulp chamber with water until all residual sealer is removed.

## CLOSE THE ACCESS {#close-the-access}

\==================================================

### STEP 99 {#step-99}

\==================================================  
CLOSE THE ACCESS with one of the following methods:  
A. Sponge and temporary restorative material.  
B. Resin bonded orifice barrier followed by a sponge and temporary restorative material.  
C. Final restoration.
